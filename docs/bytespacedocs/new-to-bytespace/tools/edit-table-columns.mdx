### Overview

The Edit Table Columns tool helps you organize and manage your data presentation effectively.

### Key Features

1. **Column Management**
   - Add/remove columns
   - Reorder columns
   - Resize and format
   - Custom column headers

2. **Data Formatting**
   - Data type specification
   - Format templates
   - Conditional formatting
   - Custom display rules

3. **Table Customization**
   - Sort options
   - Filter settings
   - Group by functionality
   - Export preferences

### Configuration Steps

1. Select table to edit
2. Customize columns
3. Set data formats
4. Save layout preferences
