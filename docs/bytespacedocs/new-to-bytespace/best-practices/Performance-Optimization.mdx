### Common Bottlenecks

<Accordion title="DOM Operations">
```javascript
// Inefficient
for (let item of items) {
  await page.click(`#item-${item}`);
}

// Optimized
const items = await page.$$('.item');
await Promise.all(items.map(item => item.click()));
```
</Accordion>

<Accordion title="Data Processing">
```javascript
// Inefficient
let results = [];
for (let page = 1; page <= totalPages; page++) {
  results.push(...await scrapeData(page));
}

// Optimized
let results = await Promise.all(
  Array.from({length: totalPages}, (_, i) => 
    scrapeData(i + 1)
  )
);
```
</Accordion>
