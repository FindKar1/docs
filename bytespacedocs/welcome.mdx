### Explore our guides and start building a Bytespace web-agent today

We're so excited to have you here! Whether you're an automation expert or you're dipping your toes in for the first time, this is your go-to resource for learning how to create reliable web automations for any website.

### Docs are intimidating, but automation doesn't have to be hard

### What you'll find here

- **Getting Started**: Learn the basics of building automations, from understanding Bytespace's automation architecture to setting up your first web-agent using our Automation Builder.
- **Using Bytespace's Visual Builder**: discover three ways to build automations and learn how to customize workflows for navigating websites, scraping data, taking actions, and managing data.
- **Templates:** explore our library of pre-built automations and learn how to use them to kickstart your building journey.
- **Best Practices:** optimize your web-agents to improve performance, ensure reliability, and build in a way that doesn't set off bot-detection alarms on websites.
- **Troubleshooting:** find solutions for common errors and tips for how to de-bug an agent if it stops working.
- **API Reference (Coming soon):** learn how to integrate Bytespace web-agents on Zapier, Make, N8N, or your own app, to extend the functionality of other automations or agents you've built.
