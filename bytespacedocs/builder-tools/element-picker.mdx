# Element Picker

The Element Picker is a powerful tool that allows you to select and interact with elements on web pages.

## Features

- Visual element selection
- Automatic selector generation
- Element property inspection
- Interactive mode

## Using the Element Picker

1. Click the Element Picker icon in the toolbar
2. Hover over elements on the page
3. Click to select an element
4. Review and adjust the generated selector
